# Groups Directory

Each group must work inside its assigned folder.

## Structure

/groups

  /group-01
  
  /group-02
  
  /group-03
  

---

## Rules

* Each group must ONLY modify their own folder
* All submissions must be done via Pull Requests
* Do NOT modify other groups' work

---

## Naming Convention

Use:

group-XX-project-name

Example:

group-01-order-system
